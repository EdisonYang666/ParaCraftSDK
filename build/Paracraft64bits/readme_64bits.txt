---+++ ParaCraft 64bits version
author: LiXizhi
date: 2015.3.17

---+++ Background 
64bits version can be installer side by side with 32bits. One just needs to put all 64bits dll and executable in a sub folder like ./bin64/ under the root install directory. 

---+++ Install Guide
   * Unzip 
   * run ParaCraft.exe to update the root folder to lastest version(32bits version will be installed)
   * run ./bin64/Paracraft_64bits.bat
   
---+++ About server mode
we can run both 32bits or 64bits in server mode, just run ParaEngineClient.exe with command line servermode="true"
see ./bin64/ParacraftServer_64bits.bat for example.
